Opdracht 2 - Functies / methodes

De console

Als je wilt testen is de console onmisbaar.

Open de console (vraag een van de mentoren eventueel om hulp).

Als dit is gelukt kun je op deze knop klikken. Er zou dan een melding moeten verschijnen in de console.

We gaan nu een functie schrijven waarmee je iets in de console kunt laten verschijnen.

Maak in functions.js een nieuwe functie aan. Deze functie noem je mijnConsoleTest. Schrijf de code om je naam in de console te laten verschijnen.
Sla je code op en laad de pagina opnieuw met de knop hiernaast: Code laden.
Klik nu op de knop hiernaast: Testen. Als het goed is verschijnt je naam in de console. Zo niet, probeer het dan opnieuw, klik weer op 'Code laden' en vervolgens weer op 'Testen'.

Je eerste functies

Laten we beginnen. Eerst gaan we een functie maken die een getal verdubbelt.

Schrijf een functie met de naam verdubbel. De functie krijg 1 parameter. De functie moet het getal verdubbelen en het antwoorden return'en.

Oké. Laten we nu wat anders proberen. We gaan nu een functie maken om de prijs van een onbekend aantal bioscoopkaartjes te bepalen.

Schrijf een functie met de naam prijsKaartjes. Deze functies krijgt 2 parameters, namelijk de prijs van een kaartje, en het aantal personen. Kies duidelijke namen voor je parameters! Bereken hoeveel iemand in totaal moet betalen.
